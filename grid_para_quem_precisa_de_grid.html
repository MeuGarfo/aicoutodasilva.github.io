Grid para quem precisa de grid

Sun 06 Jan 2019 09:03:36 PM -02

Nem sempre é necessário um grid completo, as vezes uma simples tag div com margin 0 auto e width 100% já é o suficiente.

Uma div simples com width 100% é muito mais responsiva que um grid completo que não funciona em certos browsers.

Na verdade essa div simples é adaptativa e não responsiva, mas como não há outras colunas podemos chamá-la de reponsiva.

O mais importante é lembrar de adicionar a meta tag viewport para que o texto fique legível em smartphones e tablets.
