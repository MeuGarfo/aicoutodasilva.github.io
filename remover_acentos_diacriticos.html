Remover acentos diacríticos

Wed 09 Jan 2019 07:21:00 PM -02

Caracteres com acentos diacríticos são um problema para uma pretty URL amigável.

Alguns sites não decodificam as URLs RAW mostrando os códigos alfanuméricos:

https://aicoutodasilva.github.io/ROOT_com_um_n%C3%ADvel_acima.html

Após usar uma função que "remove" os caracteres diacríticos a URL ficou assim:

https://aicoutodasilva.github.io/ROOT_com_um_nivel_acima.html

O código da função PHP para "remover" os acentos diacríticos é o seguinte:
