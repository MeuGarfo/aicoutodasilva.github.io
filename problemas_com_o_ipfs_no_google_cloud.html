Problemas com o IPFS no Google Cloud

Tue 08 Jan 2019 09:52:34 PM -02

Ao rodar o IPFS no Google Cloud tive problemas com as portas já que o IPFS usa uma para o servidor (gateway) e outra para conectar ao pool (swarm).

Para liberar o gateway desliguei o Apache e apontei a porta 8080 padrão do IPFS para a 80 do VPS.

O gateway funcionou como esperado, porém me esqueci de testar a porta do swarm e assim não pude acessar o pool.

Imagino que o pool também funcione normalmente em outra porta como a 443 que é liberada pelo firewall do Google.

Quando tiver tempo vou testar o pool em outra porta para ver se é possível rodar o IPFS plenamente no Google Cloud.
